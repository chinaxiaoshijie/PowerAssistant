# 管理助手 - 快速使用指南

## 🚀 5分钟快速启动

### 1. 克隆项目

```bash
git clone git@github.com:chinaxiaoshijie/PowerAssistant.git
cd PowerAssistant
```

### 2. 安装依赖

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 3. 配置环境

```bash
cp .env.example .env
# 编辑 .env，配置以下关键参数：
# - DATABASE_URL: PostgreSQL连接串
# - FEISHU_APP_ID: 飞书应用ID
# - FEISHU_APP_SECRET: 飞书应用密钥
# - AI_MODEL_API_KEY: AI模型API密钥（可选）
```

### 4. 初始化数据库

```bash
alembic upgrade head
```

### 5. 启动服务

```bash
# 开发模式（自动重载）
uvicorn src.main:app --reload --host 0.0.0.0 --port 8000

# 访问 Dashboard: http://localhost:8000/dashboard/
```

## 📋 核心功能使用

### 1. 查看数据同步状态

```bash
# 手动触发数据同步（在应用启动后）
curl http://localhost:8000/api/v1/feishu/sync/organization
curl http://localhost:80.0.0.0:8000/api/v1/feishu/sync/tasks
curl http://localhost:8000/api/v1/feishu/sync/projects
curl http://localhost:8000/api/v1/feishu/sync/okrs
```

**定时任务**:
- 组织架构同步：每1小时
- 任务数据同步：每2小时
- 项目数据同步：每4小时
- OKR数据同步：每6小时

### 2. 运行智能体分析

```bash
# 运行所有智能体
curl http://localhost:8000/api/v1/ai/agents/run

# 运行特定智能体
curl http://localhost:8000/api/v1/ai/agents/development_practice
curl http://localhost:8000/api/v1/ai/agents/algorithm_research
curl http://localhost:8000/api/v1/ai/agents/product_innovation
```

**智能体功能**:
- **仕杰** - 分析AI开发实践，推荐流程改进
- **运明** - 分析算法研究，评估技术应用价值
- **逍虓** - 分析产品创新，提出新功能建议
- **通用趋势** - 监控技术趋势，提供战略洞察

### 3. 生成报告

```bash
# 生成周报（上周数据）
curl http://localhost:8000/api/v1/reports/weekly?week_offset=-1

# 生成日报
curl http://localhost:8000/api/v1/reports/daily
```

**报告内容**:
- 研发健康度指标
- 任务完成统计
- 项目状态和风险
- 关键成就和挑战
- 管理建议

### 4. 查看Dashboard

访问 `http://localhost:8000/dashboard/` 可以查看：

- **概览**: 核心指标和健康度
- **任务**: 任务列表和状态
- **项目**: 项目进度和风险
- **情报**: 智能体分析结果
- **指标**: 数据可视化图表
- **报告**: 历史报告查看

## 🔔 飞书通知配置

### 1. 获取群聊ID

在飞书中：
1. 打开目标群聊
2. 点击右上角 `···` -> `群资料`
3. 复制群聊ID（格式类似 `oc_xxx`）

### 2. 配置通知接收

编辑 `src/tasks/scheduler.py`，修改第399行：

```python
# 修改前
chat_id = "oc_xxx"  # 需要替换为实际的群聊ID

# 修改后
chat_id = "oc_your_actual_chat_id_here"
```

### 3. 通知类型

系统会自动发送以下通知：

| 通知类型 | 时间 | 内容 |
|---------|------|------|
| **日报** | 每天 9:00 | AI情报日报 |
| **周报** | 每周一 10:00 | 研发管理周报 |
| **智能体分析** | 每24小时 | 仕杰/运明/逍虓的洞察 |

## 🛠️ 常见问题

### Q1: 飞书同步失败

**解决方案**:
1. 检查 `.env` 中的 `FEISHU_APP_ID` 和 `FEISHU_APP_SECRET`
2. 确认飞书应用已授权相应权限
3. 查看日志：`logs/app.log`

```bash
# 查看同步日志
tail -f logs/app.log | grep "feishu"
```

### Q2: 智能体分析无结果

**解决方案**:
1. 确保已配置AI模型API密钥
2. 检查数据库是否有情报数据
3. 手动触发情报收集

```bash
# 手动收集情报
curl http://localhost:8000/api/v1/intelligence/crawl/arxiv?limit=10
```

### Q3: Dashboard 数据为空

**解决方案**:
1. 触发数据同步
2. 等待定时任务执行（最多1小时）
3. 检查数据库连接

```bash
# 检查数据库连接
python -c "from src.database import database; print(database.engine)"
```

### Q4: 报告生成失败

**解决方案**:
1. 检查指标计算服务是否正常
2. 确认数据库中有任务和项目数据
3. 查看详细错误日志

```bash
# 查看报告生成日志
tail -f logs/app.log | grep "report"
```

## 📊 核心API接口

### 数据同步

```bash

# 同步任务
```


```bash
# 生成周报
GET /api/v1/reports/weekly?week_offset={-1}

# 获取历史报告
GET /api/v1/reports/history
```

```bash

# 获取任务列表
GET /api/v1/tasks?status={all|completed|in_progress}

# 获取项目列表
GET /api/v1/projects

# 获取情报数据
GET /api/v1/intelligence/items?category={all|algorithm|product|...}
```

## 🎯 开发建议

### 1. 本地开发

```bash
# 启用热重载
uvicorn src.main:app --reload

# 启用详细日志
export LOG_LEVEL=DEBUG
```

### 2. 数据库迁移

```bash
# 创建新迁移
alembic revision -m "描述你的更改"

# 应用迁移
alembic upgrade head

# 回滚迁移
alembic downgrade -1
```

### 3. 运行测试

```bash
# 运行所有测试
pytest

# 运行特定测试
pytest tests/services/ai_intelligence/test_agents.py -v

# 生成覆盖率报告
pytest --cov=src --cov-report=html
```

### 4. 调试技巧

```python
# 在代码中添加调试日志
import structlog
logger = structlog.get_logger()
logger.debug("debug_message", key=value)

# 查看日志
tail -f logs/app.log
```
