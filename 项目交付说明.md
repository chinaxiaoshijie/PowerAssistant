## 🚀 快速启动

### 1. 克隆仓库
```bash
git clone git@github.com:chinaxiaoshijie/PowerAssistant.git
cd PowerAssistant
```

### 2. 安装依赖
```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 3. 配置环境
```bash
cp .env.example .env
# 编辑 .env 文件，配置：
# - 数据库连接信息
# - 飞书 APP_ID 和 APP_SECRET
# - AI模型 API密钥
```

### 4. 初始化数据库
```bash
alembic upgrade head
```

### 5. 启动服务
```bash
uvicorn src.main:app --reload --host 0.0.0.0 --port 8000
```

### 6. 访问系统
- 🎨 **Dashboard**: http://localhost:8000/dashboard/
- 📚 **API文档**: http://localhost:8000/docs
- 🔴 **API Redoc**: http://localhost:8000/redoc

## 🔧 配置飞书集成

### 1. 创建飞书应用
- 登录 [飞书开放平台](https://open.feishu.cn/)
- 创建企业自建应用
- 获取 `APP_ID` 和 `APP_SECRET`

### 2. 配置权限
申请以下权限：
- `contact:user.read` - 读取用户列表
- `task:task:read` - 读取任务数据
- `docs:doc:read` - 读取文档内容
- `okr:okr:read` - 读取OKR数据

### 3. 配置群聊推送
编辑 `src/tasks/scheduler.py`，修改第399行的 `chat_id` 为实际的飞书群聊ID：
```python
chat_id = "oc_your_actual_chat_id_here"
```

### 4. 配置到环境变量
```bash
FEISHU_APP_ID=your_app_id
FEISHU_APP_SECRET=your_app_secret
```
