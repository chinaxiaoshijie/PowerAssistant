## 核心功能模块

### 1. 飞书数据同步 (`src/services/feishu/`)
- 组织架构同步（部门/用户）
- 任务数据拉取
- 项目进度同步
- 文档内容抓取
- 自动 Token 管理 + 错误重试

### 2. 研发健康度指标 (`src/services/metrics/rnd_health.py`)
7 个核心指标：整体健康度、模块成熟度、任务准时完成率、技术债浓度、任务延期率、单点依赖风险、研发保护时间

### 3. 交付健康度指标 (`src/services/metrics/delivery_health.py`)
5 个核心指标：整体健康度、交付准时率、版本成功率、客户问题率、实施回滚风险

### 4. 报告生成服务 (`src/services/report/report_generation.py`)
- 周报/月报自动生成（Markdown 格式）
- 智能成就/挑战识别
- AI 推荐生成（DashScope，带规则 fallback）
- 飞书卡片消息推送

### 5. AI 情报抓取 (`src/services/ai_intelligence/`)
- 信息源：arXiv、GitHub Trending、HackerNews、Reddit、RSS 博客
- 内容自动分析与相关性评分
- 定时抓取调度

### 6. AI 决策引擎 (`src/services/ai_engine/`)
- DashScope（通义千问）Provider
- 多模型路由策略
- 可配置 AI 接口

### 7. API 接口层 (`src/api/v1/`)
- RESTful 接口（FastAPI）
- 指标查询、报告生成、同步触发、组织架构、Dashboard

### 8. 定时任务调度 (`src/tasks/`)
- APScheduler 异步任务
- 飞书数据定时同步
- 情报定时抓取

## 项目结构

```
src/
├── api/v1/           # FastAPI 接口层
├── config/           # 配置（settings, ai_settings）
├── models/           # SQLAlchemy 数据模型
├── schemas/          # Pydantic 请求/响应模型
├── services/
│   ├── feishu/       # 飞书数据同步
│   ├── ai_engine/    # AI 引擎
│   ├── ai_intelligence/  # 情报抓取
│   ├── metrics/      # 健康度指标计算
│   └── report/       # 报告生成
└── tasks/            # 定时任务

tests/
├── unit/             # 单元测试（91 个用例）
└── integration/      # 集成测试（12 个用例）

docs/                 # 技术文档
alembic/              # 数据库迁移
```
