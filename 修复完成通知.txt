📊 修复详情

问题描述:
  ❌ ERROR: function json_contains(json, character varying) does not exist

根本原因:
  使用了 PostgreSQL 不支持的 JSON 函数

解决方案:
  ✅ 改用 PostgreSQL 原生的 '?' 操作符
  ✅ 添加缺失的 'or_' 导入

代码变更:
  文件: src/services/metrics/delivery_health.py

  修改前:
    func.json_contains(FeishuTask.labels, "bug")

  修改后:
    FeishuTask.labels.op("?")("bug")

💡 技术要点

PostgreSQL JSON 操作符:
  • ?  - 检查键或值是否存在
  • @> - 包含 (右侧在左侧中)
  • -> - 提取字段
  • ->> - 提取字段为文本

SQLAlchemy 用法:
  FeishuTask.labels.op("?")("bug")
